# sizzletracker documentation

Documentation for **sizzletracker**, a terminal-based MIDI tracker / step
sequencer. The project is released under the **GPL-3.0** (see
[`../LICENSE`](../LICENSE)).

| Document | For whom | What it covers |
|---|---|---|
| [USER_GUIDE.md](USER_GUIDE.md) | Musicians / end users | Install, a first-song tutorial, every editing surface, MIDI setup, recording, saving/exporting, and a keyboard cheat-sheet. |
| [DEVELOPER_GUIDE.md](DEVELOPER_GUIDE.md) | Contributors / maintainers | A from-scratch handover: architecture, data model, concurrency, file formats, build, testing, and how to extend the code. |
| [LICENSING.md](LICENSING.md) | Release / legal | The GPL-3.0 compatibility analysis, dependency license inventory, obligations, and alternatives. |
| [THIRD_PARTY_NOTICES.md](THIRD_PARTY_NOTICES.md) | Distribution | Attribution and license notices for bundled and linked components. |

For a quick overview and build instructions, see the top-level
[`../README.md`](../README.md).
